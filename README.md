Spectre.css and React integration

## Usage

Installation:

```sh
npm install spectre-react-lib
```

Importing components:

```js
import {
  Page,
  Pagination,
  Panel,
  Tab,
  Toast,
  Button,
  Table,
  Grid,
  Divider,
  Loading,
  Parallax,
  ControlledTab,
  EmptyState,
} from 'spectre-react-lib';
```
