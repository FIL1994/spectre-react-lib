Spectre.css and React integration

## Usage
Installation:
```
npm i -S spectre-react-lib
```

Importing components:
```javascript
import {
  Page,
  Pagination,
  Panel,
  Tab,
  Toast,
  Button,
  Table,
  Grid,
  Divider,
  Loading,
  Parallax,
  ControlledTab,
  EmptyState
} from "spectre-react-lib";
```
